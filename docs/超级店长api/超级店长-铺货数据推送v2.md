# 请求


| 字段     | 类型   | 是否必须 | 示例 | 描述                                   |
| -------- | ------ | -------- | ---- | -------------------------------------- |
| date     | String | 是       |      | 数据时间，格式:"yyyy-MM-dd HH:mm:ss" |
| userId   | String | 是       |      | 批批⽹⽤户id                           |
| goodsId  | String | 是       |      | 批批⽹商品id                           |
| platform | String | 是       |      | 绑定的下游平台，多个平台以,分隔，例如PDD,DY                        |
| price    | String | 是       |      | 铺货价格，例如39.0~47.0 单位元         |
